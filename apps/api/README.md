# API (AdonisJS)

Ce dossier contient la base backend AdonisJS :

- routes API pour portfolio/blog/contact,
- contrôleurs HTTP,
- services (Resend),
- validateurs,
- migrations SQL compatibles PostgreSQL.

> Note: le scaffold est volontairement "source-first" pour rester compatible avec un environnement sans accès npm complet. Dès que l’accès registre est disponible, installer AdonisJS et brancher ces fichiers au runtime officiel.
